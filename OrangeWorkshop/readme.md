# wogame.co 机场订阅自动获取
## 什么是ogame.co：

> 一个还蛮好用的机场，买了11块钱的用了几个月，很好用。但是订阅地址经常更新，导致需要自己修改订阅就很烦。这个爬虫自动获取最新订阅地址并转发。将这个脚本运行在你的服务器上后，将订阅地址修改为下面教程的地址即可永久自动获取最新订阅


#  使用教程：
<table><tr><td bgcolor=orange>-----------------------------</td></tr></table>

<font color=red>运行main.py：</font>[run.bat](file:./run.bat)

<font color=red>新的订阅地址为：</font>[http://127.0.0.1:8000/?emai=你的邮箱&pass=密码](http://127.0.0.1:8000/?emai=你的邮箱&pass=密码)
