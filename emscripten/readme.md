# Emscripten环境配置
## 什么是emscripten：

> Emscripten 是一个用于编译为 asm.js 和 WebAssembly 的工具链，使用 LLVM 构建，可让您在 Web 上以接近本机的速度运行 C 和 C++，而无需插件。
